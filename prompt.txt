1. "Devo realizzare un'applicazione gestionale con Node.js ed Express per monitorare le scorte di un magazzino alimentare. L'app deve servire file statici dalla cartella public e gestire rotte API per il recupero e l'aggiornamento dei dati. Mi potresti scrivere lo scheletro del file server.js includendo le importazioni per fs, csv-parser e xml2js, e spiegarmi i passaggi per inizializzare il progetto con npm installando le dipendenze necessarie?"

2. "Ora dobbiamo popolare il magazzino creando 3 file nella cartella /data: magazzino.json, magazzino.csv e magazzino.xml. Ogni file deve contenere 6 prodotti alimentari con i campi prodotto, categoria e quantita. Per il file XML, assicurati di usare la struttura <magazzino><item> richiesta dal parser nel server. Vorrei una varietà di categorie come Latticini, Pasta e Ortofrutta."

3. "Nel file server.js, ho bisogno di implementare la funzione loadData() che aggreghi i dati dai tre formati in un unico array fullInventory. Come posso gestire correttamente l'asincronia affinché il server attenda la fine della lettura dello Stream CSV e il completamento di xml2js? Inoltre, ogni oggetto aggiunto all'array deve includere una proprietà fonte (es. 'JSON', 'XML', 'CSV') per permettere al frontend di distinguerne l'origine."

4. "In index.html, devo implementare la funzione getInventory() che recupera i dati tramite fetch('/api/magazzino'). Come posso generare dinamicamente le card all'avvio della pagina, includendo una logica che assegni la classe CSS .low-stock se la quantità è inferiore a 25 unità? Inoltre, come posso gestire la creazione di pulsanti differenti (Incrementa o Compra) in base al valore userRole salvato nel localStorage?"

5. "Per completare l'app, mi serve un file style.css che supporti le classi definite nel frontend, come .movie-grid, .card e .source-tag. Vorrei un design moderno con toni beige e bianchi, un layout a griglia responsiva e degli effetti hover sulle card. Potresti includere anche lo stile per i badge delle fonti (JSON, CSV, XML) e per la segnalazione visiva di 'Scorta Bassa', assicurandoti che i pulsanti di login e logout siano coerenti con l'interfaccia?"
